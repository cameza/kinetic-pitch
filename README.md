# Kinetic Pitch - PFF Team Generator

This repository contains the PFF Team Generator project, including logic for player management, team drafting, and design mockups.

## Project Structure

- **/pff-app**: The core Next.js application.
- **/gridiron_neon**: Design assets and themes for the "Gridiron Neon" aesthetic.
- **/confirmation_updated**, **/home_screen_updated**, **/player_selection_updated**, **/teams_generated_updated**: UI design mockups and reference code for various app screens.

## Getting Started

To run the application, navigate to the `pff-app` directory:

```bash
cd pff-app
npm install
npm run dev
```

The app will be available at `http://localhost:3000`.
